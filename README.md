# Kaggle Dataset Loader + Prediction Model

## About
This repo provides Python utilities to download and load Kaggle datasets as pandas DataFrames.  
It also includes a TensorFlow model that predicts answers to user-prompted questions about uploaded data.  
An MCP server integration will be added soon.

## Setup (uv)

### 1) Install uv
```bash
pip install uv
uv venv

macOS/Linux
source .venv\bin\activate

Windows
.venv\Scripts\activate

uv sync

python main.py
```
